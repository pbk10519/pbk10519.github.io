# 🎄 AI로 자라는 아이(AI) - 교육용 웹 게임 모음

초등학교 학급 운영과 학습을 위한 인터랙티브 웹 애플리케이션 모음입니다. Claude AI를 활용하여 제작되었으며, 학급 관리 도구와 교육용 게임을 제공합니다.

## 📚 프로젝트 소개

이 프로젝트는 초등학교 교실에서 활용할 수 있는 다양한 웹 기반 도구와 게임을 제공합니다. 학급 운영의 효율성을 높이고, 학생들이 즐겁게 학습할 수 있는 환경을 만드는 것을 목표로 합니다.

## 🎮 주요 기능

### 학급 운영 도구

#### 1. **자리배치 시스템** (`seatingarrangement.html`)
- 학생 수를 입력하고 랜덤으로 자리를 배정
- 드래그 앤 드롭으로 자유롭게 자리 조정
- 6열 교실 레이아웃 지원

#### 2. **1인 1역 랜덤 뽑기** (`role.html`)
- 학급 역할을 공정하게 배정
- 이전 역할과 중복을 최소화하는 스마트 알고리즘
- 25가지 기본 역할 제공 (확장 가능)

#### 3. **상품 뽑기** (`candy.html`)
- 재미있는 보상 시스템
- Tone.js를 활용한 효과음
- 긴장감 넘치는 애니메이션

### 학습 게임

#### 4. **구구단 할리갈리** (`99.html`)
- 2-3명 플레이 가능
- 과일 카드를 뒤집어 5개가 되면 종 치기
- 구구단 문제를 풀어야 득점

#### 5. **알파벳 뱀 주사위 게임** (`alphabet.html`)
- 1-3명 플레이 지원
- 뱀과 사다리가 있는 보드게임
- 알파벳 퀴즈를 풀어야 이동 가능
- 6×6 보드 (36칸)

#### 6. **맞춤법 갤러그** (`spelling.html`)
- 슈팅 게임 형식의 맞춤법 학습
- 떨어지는 블록을 파괴하거나 퀴즈 풀기
- 생명 시스템과 점수 기록
- 로서/로써, 왠지/웬지, 돼/되 등 헷갈리는 맞춤법 학습

## 🛠️ 기술 스택

- **HTML5** - 구조
- **Tailwind CSS** - 스타일링
- **Vanilla JavaScript** - 게임 로직
- **Canvas API** - 게임 렌더링 (맞춤법 갤러그)
- **Tone.js** - 오디오 효과 (상품 뽑기)
